# Equal AI Claude Skills

Claude Code plugin marketplace for the Equal AI team.

## Installation

```bash
/plugin marketplace add inifity-tech/equal-ai-claude-skills
```

## Available Plugins

List available plugins:

```bash
/plugin list equal-ai-claude-skills
```

Install a plugin:

```bash
/plugin install <plugin-name>@equal-ai-claude-skills
```

## Updating

```bash
/plugin marketplace update equal-ai-claude-skills
```
